Bibiloteca deSolve

R, como ya saben, es un lenguaje de programación extremadamente versátil. Con R, pueden realizar una amplia variedad de tareas, como generar gráficos, realizar análisis estadísticos, analizar secuencias biológicas, llevar a cabo alineamientos, construir árboles filogenéticos, crear redes, y analizar datos de secuenciación de nueva generación, incluyendo RNA-Seq, microbioma, epigenética, entre otros.

Atractor de Lorenz

Por ejemplo, un caso clásico de ecuaciones diferenciales que modelan un sistema complejo es el conjunto de ecuaciones de Lorenz, que describen el comportamiento caótico de la atmósfera. Estas ecuaciones son un ejemplo emblemático de cómo un sistema dinámico puede exhibir caos, incluso bajo condiciones deterministas. Para ver una implementación en R de estas ecuaciones y la visualización del atractor de Lorenz, puedes seguir este enlace a un ejemplo en RMarkdown: Visualización del Atractor de Lorenz.

Este ejemplo ilustrará cómo, con solo unas pocas líneas de código, es posible modelar y visualizar el comportamiento caótico en un sistema dinámico, proporcionando una poderosa demostración de las capacidades de R para manejar problemas complejos en la ciencia y la ingeniería.

\[\dot x = a x +yz\] \[\dot y = b(y-z)\] \[\dot z = -xy+cy-z \]

## Chaos in the atmosphere
## Definimos una función que tiene las ecuaciones
Lorenz <- function(t, state, parameters) {
with(as.list(c(state, parameters)), { 
       # Acá van definidas las ecuaciones
       dX<- a*X+Y*Z
       dY<- b*(Y-Z)
       dZ <- -X * Y + c * Y - Z
        list(c(dX, dY, dZ))
      })
}
# Acá le damos valores a los parámetros
parameters <- c(a = -8/3, b = -10, c = 28)
#Estas son las condiciones iniciales
state <- c(X = 1, Y = 1, Z = 1)
#En este vector ponemos el tiempo para evaluar 
times <- seq(0, 100, by = 0.01)
#Acá hace todo (ode significa *ordinary differential equation*)
out <- ode(y = state, times = times, func = Lorenz, parms = parameters)
#Graficamos las soluciones, una gráfica por cada cooordenada (variable)
plot(out,col="blue")

scatterplot3d(out[,-1], type = "l")

Lotka-Volterra

Uno de los ejemplos clásicos de un sistema dinámico modelado por ecuaciones diferenciales es el modelo de Lotka-Volterra. Este modelo describe las interacciones depredador-presa y es un punto de partida fundamental en la teoría ecológica y la dinámica de poblaciones. El modelo más simple de Lotka-Volterra se compone de dos ecuaciones diferenciales:

\[\frac{d\ Presa}{dt} = a*Presa-b*Presa*Depredador\] \[\frac{d\ Depredador}{dt} = -c*Depredador+ d*Presa*Depredador\]

#Acá se define la función que tiene las ecuaciones diferenciales
LotkaVolterra <- function(Time, State, Pars) { 
  with(as.list(c(State, Pars)), {
dPresa <- a*Presa -b*Presa*Depredador
dDepredador <- d*Presa*Depredador -c*Depredador
return(list(c(dPresa, dDepredador)))
})
}

Después se definen los parámetros, el tiempo de integración, la integración y las gráficas

SIR sin demografía

Ahora sí podemos resolver las ecuaciones del modelo SIR sin demografía. Las ecuaciones son las ya conocidas

\[ \dot S = -\beta SI \] \[\dot I= \beta SI -\gamma I\]

\[\dot R = \gamma I\]

con \[R_0=\frac{\beta}{\gamma}\]

Acá definimos la función que resuelve las ecuaciones

SIR <- function(t, state, parameters) {
with(as.list(c(state, parameters)), { 
       # Acá van definidas las ecuaciones
       # Vamos a suponer dependencia de la frecuencia
       dS<- -beta*S*I/(S+I+R)
       dI<- beta*S*I/(S+I+R) -gama*I
       dR<- gama*I
        list(c(dS, dI, dR))
      })
}
pars <- c(beta= 4, gama = 2 )  
condiciones_iniciales <- c(S =999, I=1,R=0)
tiempo <- seq(0, 20, by = 0.001)
out <- ode(condiciones_iniciales, tiempo, SIR, pars)

Ahora sí graficamos

matplot(out[ , 1], out[ , 2:4], type = "l", xlab = "tiempo", ylab = "Población",
main = "SIR", lwd = 2)
legend("topright", c("Susceptible", "Infectado","Recuperado"), col = 1:3,lty=1:3,cex=0.5)

Auto inhibición

\[\dot x= \frac{\beta k^n}{k^n +x^n} -\alpha x\]

\[1 \leq n \leq 4\] \[\beta >0\] \[0 \leq k \leq \beta\] \[\alpha >0\]

auto_inh <- function(t, state, parameters) {
    with(as.list(c(state, parameters)), { 
           # The list of equations
           dX <- ((beta*(k)^n)/((k^n)+(x^n)))-alpha*x
            list(c(dX))
          })
    }
    # Paramaters values
    parameters <- c(k=1, alpha=5, n=1, beta=1)
    
    # Initial conditions
    state <- c(x=0)
    
    # The time lapse we'll be using
    times <- seq(0, 100, by = 0.01)
    
    # we make the differential equation "ode"
    out <- ode(y = state, times = times, func = auto_inh, parms = parameters)
    
    # we graph
    plot(out,col="blue")

    tail(out)
